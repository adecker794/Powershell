What these sets of scripts do:
It goes to each machine in a list and pulls the Mac Adresses of all network cards.


Where to place scripts:
c:\temp\scripts\getmac


What to change:
c:\temp\scripts\getmac\computers.txt - add in your list of computers

Output:
c:\temp\scripts\getmac\macs.csv - it is a list of all macs obtained from the list of computers


How to run:
Open in ISE and run(after defining computers.txt)
Or just run from CLI
